### 套路之快慢指针

由于链表不知道其长度且是否有环，快慢指针在很多场景得到价值体现。


## leetcode 202.快乐数

咋一看题目描述跟链表毫无相干，但是实际上用到了链表的 `唯一指向性` 的思想。每个数对应的下一个数都是唯一的，所以只需要判断是不是一个环，如果不是环就最终结果到1，也就是快乐数

不用担心数字会越来越大，即使范围内可得到最大的数是：1999999999 => 9*81+1 = 730s
